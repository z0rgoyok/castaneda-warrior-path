# Castaneda Anti-Anxiety App

Проект по анализу книг Карлоса Кастанеды и трансформации практик пути воина в современное анти-тревожное приложение.

## Цель

Извлечь из учения Дона Хуана практические техники работы с тревогой, страхом и внутренним диалогом, адаптировав их для современного пользователя в формате мобильного/веб-приложения.

## Ключевые концепции для исследования

### Из учения Дона Хуана

- **Остановка внутреннего диалога** — основа многих тревожных состояний
- **Контролируемая глупость** — отстранённое отношение к происходящему
- **Стирание личной истории** — освобождение от груза прошлого
- **Смерть как советчик** — переоценка значимости проблем
- **Путь сердца** — выбор деятельности без сожалений
- **Сталкинг** — отслеживание своих паттернов поведения
- **Перепросмотр** — техника работы с эмоциональными зарядами прошлого
- **Не-делание** — выход из автоматических реакций

## Структура проекта

```
├── data/           # Исходные тексты (fb2)
├── research/       # Заметки по анализу, выписки, концепции
├── docs/           # Документация продукта, user stories
└── src/            # Прототипы, код приложения
```

## Этапы работы

1. **Анализ текстов** — выделение практик и техник
2. **Категоризация** — группировка по типу тревоги/проблемы
3. **Адаптация** — перевод на современный язык без эзотерики
4. **Дизайн UX** — как подавать практики пользователю
5. **Прототип** — MVP приложения
